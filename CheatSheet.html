WorldWideWeb

The technologies that made the world wide a succes where HTML and the elements it works with:
Hypertext/Hypermedia (which is are just texts, images and audios with Hyperlinks), Hyperlinks (which are just clickable refrences to another file )
Hypermedia and other Hypertext formats are the most important resources of HTML
a URL is made of a

- scheme/protocol
- (optional) infos about the host
- the path to the document
- (optional) a query with further data
- (optional) a fragment refers to another source on the webserver

WebServer

A webserver is a server which has webserver software installed, stores webpages and lets clients request or upload files
It's stores its content on a normal file path on some drive, and the content mostly consist of Html files and other media

WebClient

A web client is a software which allows users to acces webservers and request/upload files
And a web client is Not the same as a WebBrowser ,because WebBrowsers also have some WebServer attributes.
